## 🛠️ CUSTOMIZATION GUIDE

### Easy Customizations

#### 1. Change Colors
Edit CSS variables in `style.css`:
```css
:root {
    --color-accent: #ff69b4;      /* Change accent color */
    --color-lily-pink: #ffb6d9;   /* Change primary pink */
    --color-lily-purple: #d8a8f0; /* Change purple accent */
}
```

#### 2. Adjust Animation Speed
Modify transition timing:
```css
--transition-fast: 0.2s cubic-bezier(0.23, 1, 0.320, 1);
--transition-base: 0.4s cubic-bezier(0.23, 1, 0.320, 1);
--transition-slow: 0.8s cubic-bezier(0.23, 1, 0.320, 1);
```

#### 3. Customize Letter Content
Edit HTML in `index.html`:
- Update title in `.letter-title`
- Modify paragraphs in `.letter-paragraph`
- Change song info in `.song-info`

#### 4. Change Fonts
Replace font families in CSS:
```css
--font-primary: 'Poppins', sans-serif;      /* Body font */
--font-secondary: 'Playfair Display', serif; /* Display font */
```

#### 5. Add Music
Place MP3 file at `assets/song.mp3` and update song info

### Advanced Customizations

#### Add New Animations
Create keyframe in CSS:
```css
@keyframes customAnimation {
    from { /* starting state */ }
    to { /* ending state */ }
}
```

#### Modify Classes
Create new class-based modules in JavaScript:
```javascript
class CustomFeature {
    constructor() {
        this.init();
    }
    init() {
        // Initialize feature
    }
}
```

#### Enhanced Effects
Leverage existing systems:
- `CursorManager` - Custom cursor logic
- `PetalsAnimationSystem` - Floating elements
- `ButtonEffects` - Interactive states
- `MusicPlayerSystem` - Audio controls

## 🐛 TROUBLESHOOTING

### Cursor Not Showing
- Check if JavaScript is enabled
- Verify DOM references are correct
- Check browser console for errors

### Animations Not Smooth
- Update browser to latest version
- Disable browser extensions
- Check GPU acceleration is enabled
- Clear browser cache

### Music Not Playing
- Ensure `assets/song.mp3` exists
- Use local server (not direct file)
- Check browser console for errors
- Try different audio format

### Colors Look Different
- Check browser color profile
- Try different browser
- Verify CSS file is loading
- Clear browser cache
