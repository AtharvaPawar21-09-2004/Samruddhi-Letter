## ✏️ PERSONALIZE (2 minutes)

### Change the Letter
1. Open `index.html` in any text editor
2. Find `<div class="letter-body">`
3. Edit the paragraphs
4. Save and refresh
5. Your custom message appears!

### Change Colors
1. Open `style.css`
2. Find `:root {` at the top
3. Change these colors:
   ```css
   --color-accent: #ff69b4;           /* Main pink */
   --color-lily-pink: #ffb6d9;        /* Light pink */
   --color-lily-purple: #d8a8f0;      /* Purple tone */
   ```
4. Save and refresh
5. New colors applied instantly!

### Change Song Title
1. Open `index.html`
2. Find `<p class="song-title">Our Journey</p>`
3. Replace "Our Journey" with your song name
4. Save and refresh

## 🛠️ TROUBLESHOOTING

### Music doesn't play
```
✓ Check if assets/song.mp3 exists
✓ Use local server (not direct file open)
✓ Refresh page (Ctrl+R or Cmd+R)
✓ Try a different browser
```

### Website looks wrong
```
✓ Hard refresh (Ctrl+Shift+R or Cmd+Shift+R)
✓ Clear browser cache
✓ Check if CSS file loaded (right-click → Inspect)
```

### Animations are jerky
```
✓ Close other browser tabs
✓ Update your browser
✓ Disable browser extensions
✓ Check GPU acceleration is on
```

### Can't find Open button
```
✓ It's in the center of the screen
✓ It says "Open When You're Ready 💌"
✓ It glows on hover
✓ Refresh page if you don't see it
```
